Leveraging Multiple (Less-Trusted) Sources to Improve Metabolic Pathway Prediction
Abdur Rahman M. A. Basher, Steven J. Hallam
Under preparation, 2020
bioRxiv / code

This paper presents mltS (multi-label learning from less-trusted sources) that leverages the idea of estimating each model's reliability score in the presence of small reference collection dataset.

reMap: Relabeling Metabolic Pathway Dataset with Bags to Enhance Predictive Performance
Abdur Rahman M. A. Basher, Steven J. Hallam
Under review, 2020
bioRxiv / code

This paper presents the reMap framework that performs relabeling examples to a different set of labels, characterized as bags that may be considered a fresh clean label set, where a bag is comprised of correlated original label set.
